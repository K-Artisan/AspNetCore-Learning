>
> ASP.NET Core Identity的角色不是对用户进行身份验证，而是对其进行管理。但是扩展可以做的是进一步执行此步骤，并添加诸如对用户进行身份验证之类的功能。那么回头看这个问题，为什么许多开发人员将ASP.NET Core Identity视为身份验证库？由于它是提供此类功能的广为人知的常用扩展
